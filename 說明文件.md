# LnData Web Engineer Pre-Test

## Task1 LnData-Nodejs

1. 下載LnData-Nodejs的壓縮檔案、開啟檔案後請先在終端機下指令： npm i 
2. 安裝完成後請到.env 檔 內修改DB_USER, DB_PASS 為本機端的帳號密碼，並確認Mysql內有company的資料庫
3. 設定完成請在終端機下指令：nodemon 開啟服務、即可在瀏覽器中輸入URL查詢資料
